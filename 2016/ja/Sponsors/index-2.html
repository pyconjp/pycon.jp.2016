PyCon JP のスポンサーについて

Diamond

株式会社フンザ

http://hunza.jp/

フンザは「世の中の文化となるウェブサービスを創る」をビジョンに、国内No.1のC2Cチケット売買サイト「チケットキャンプ」を開発/運営しています。

Platinum

MonotaRO

https://www.monotaro.com/

「ITで、間接資材調達を変革する」
モノタロウは、働く現場で必要となる様々な間接資材(最終製品となる原材料を除く全ての資材)約900万点をインターネットで販売しています。
様々な現場のニーズにお応えすべく、自社開発の高度な検索システムと精緻なデータベースマーケティングが実現する「お客様ごとの最適化したレコメンドサービス」で、従来の非効率的な間接資材調達を変革し、社会に新しい価値を提供しています。

Gandi.net

https://www.gandi.net/

GANDI (www.gandi.net) は.COM, .JP, .IO, .MOE等700以上の拡張子の革新的なドメイン名取得業者、DNS提供者です。
私たちはスタートアップから有名大手企業まで幅広いお客様の約200万ものドメイン名を管理しています。
Gandiは1999年にフランスで創立され、現在はサンフランシスコ、パリ、ルクセンブルク、台北にオフィスを構えています。

Gold

株式会社JX通信社

http://jxpress.net/

「技術で、ビジネスとジャーナリズムを両立させる」
JX通信社は国内で唯一、共同通信社をはじめとするマスコミ各社と連携してテクノロジーを開発し、提供している会社です。
消費者向けのサービスとしてはニュース地震速報アプリ「NewsDigest」などを開発しています。国内最速でニュースや災害速報を届けることもある高い技術力で信頼を得ています。
マスコミだけでなく企業のポータルサイトなどにも広く活用できる技術力で、今後も様々なサービス展開を予定しています。

Port

https://www.theport.jp/

先日、シリーズBで9億円の資金調達を完了した、創業6年目の成長ベンチャーであるPORT INC.のミッションは、「世界中に、アタリマエとシアワセを」です。そのために、「テクノロジー×リアル産業」にドメインを置き、ビックビジネスを仕掛ける会社です。世の中に「あったらいいな」ではなく「なくてはならない」といわれる、革新的なプロダクトを生み出すことで、社会課題に対して、「提言」ではなく、「実行」する会社です。

株式会社HDE

https://www.hde.co.jp/

クラウドセキュリティ・Eメール配信に関する SaaS/PaaS を企業向けに提供しています。

コンセプトは「テクノロジーの開放」。
私たち独自の技術や、時代の先端を行く技術を広くお客様に届け、世の中を変えてゆくことを意味します。
私たちはITの可能性を強く信じています。ITが世界を変えてきたこと、今なお世界を変え続けていること、その驚くべき発展のスピードに、畏敬の念を抱いています。
同時に私たちは、素晴らしいテクノロジーが、時として人々の手に入りにくい形で出現することも知っています。ITの力を享受するためには、誰かが理想と現実のギャップを埋める必要があるのです。
私たちは、テクノロジーにおけるこのギャップの橋渡し役として、お客様に新しい価値を提供し続け、世界の発展に貢献します。

株式会社いい生活

http://www.e-seikatsu.info/

「不動産テックで世界を変える」
人は住むことをやめられません。
社会生活の基盤として、全ての人が一生付き合い続けなければならない不動産。
いい生活は、不動産業界をテクノロジーの力でオープンな世界へと変革し、社会に貢献することをビジョンとしています。
主力プロダクトである「ESいい物件One」は不動産にかかわる様々な業務を網羅するサービスで、2015年にはASPICクラウドアワードでグランプリも獲得しました。
その中核となるサーバAPI群は全てPythonで実装されています。

株式会社白ヤギコーポレーション

https://shiroyagi.co.jp/

白ヤギコーポレーションでは、「深く知ることで生きることを豊かにする」というビジョンを実現するため、ニュースキュレーションアプリ「カメリオ」や、そのエンジンの法人向けAPI「カメリオAPI」に加え、データ分析の教育サービスである「白ヤギアナリティクス」を展開しています。

白ヤギコーポレーションでは、カメリオのバックエンドを支えるPythonエンジニアや、アルゴリズムの精度を上げるデータサイエンティストを常時募集しています。

1993年以来、NVIDIA（NASDAQ: NVDA）は、ビジュアル・コンピューティングという芸術的な科学の世界をリードしてきました。ゲーミング、自動車、データセンターおよびプロフェショナル・ビジュアリゼーションの分野で特化したプラットフォームを提供し続けています。NVIDIAの製品は仮想現実、人工知能、自律走行車の開発においても最新の技術を提供しています。詳しい情報は、http://nvidianews.nvidia.com/をご覧ください。

https://azure.microsoft.com/ja-jp/

JetBrainsは、開発者の方々の手腕を問われるタスクを簡素化し、ルーチンを自動化し、開発者の方々が最高の開発プラクティスを採用するご支援をすることで、開発者の方々の生産性を高めるインテリジェントなソフトウェアソリューションをご提供しています。PyCharmは、JetBrainsによるプロフェッショナルな開発者の方々向けのPython IDEで、生産的なPython、ウェブ、および科学的な開発用の完全なツールセットをご提供いたします。

Python Software Foundation

http://python.org/psf

The mission of the Python Software Foundation is to promote, protect, and advance the Python programming language, and to support and facilitate the growth of a diverse and international community of Python programmers. Being part of the PSF means being part of the Python community. The PSF is an open membership organization so that everyone who uses and supports Python can join.

dots.

http://eventdots.jp/

面白そうな勉強会が簡単に見つかればいいのに！という思いから「dots.（ドッツ）」が生まれました。dots.では、ITに関わる様々なイベント・勉強会・講演会・交流会・カンファレンス・セミナーなどの情報を集約し掲載しています。渋谷駅徒歩３分の立地に最大200名が着席できるイベント&コミュニティースペース「dots.(https://eventdots.jp/space)」を構え、オンラインとオフラインの両方で技術情報が取得できるサービスです。

DATUM STUDIO

https://datumstudio.jp/

DATUM STUDIO は、データを活用しようとするすべての企業を、人工知能（AI）を通し支援する会社です。どのデータと、どのツールを、どのように使えば、人工知能ビジネスの中で価値を生むかについて、お客様の状況とニーズに合わせ一緒に検討していきます。人工知能のカスタマイズ、Webクローラの作成、データ分析基盤の構築、その他、社員研修、分析コンサルティングなど、お客様ごとに最適な支援を行います。特に人工知能に基づく、データマイニング・パターン抽出領域において豊富な経験・技術力を有しており、お客様のデータにあわせた、ビジネス活用を支援する人工知能のカスタマイズ開発に強みがあります。
